# Ubuntu20.04 ARM64编译与安装docker
- 瑞芯微提供的sdk，版本是 RK3588-linux-Beta-202204151412.tar
- 开发板：rk3588 
- 内核版本 5.10.66
- sdk：RK3588-linux-Beta-202204151412
- 编译系统：Ubuntu20.04-22.04
- rootfs：Ubuntu20.04 ARM64位 暂时不提供源代码了
